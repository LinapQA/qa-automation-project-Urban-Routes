# **Proyecto: Pruebas Automatizadas para pedir un taxi en Urban Routes**

### Descripción del Proyecto

Este proyecto se centra en el desarrollo de pruebas automatizadas para cubrir el proceso completo de pedir un taxi a través de una aplicación Urban Routes. Las pruebas abarcan acciones como configurar la dirección, seleccionar la tarifa Comfort, agregar información de contacto y pago, enviar mensajes al conductor, solicitar artículos adicionales como mantas y pañuelos, así como verificar la aparición de un modal para buscar un taxi.

### Tecnologías y Técnicas Utilizadas

* Selenium WebDriver:Se utiliza para interactuar con la aplicación web y realizar las pruebas automatizadas.

* Python: El lenguaje de programación principal utilizado para escribir el código de las pruebas y el script principal.

* Page Object Model (POM): Se emplea para organizar el código de las pruebas, separando las acciones de las páginas web en clases dedicadas.

* WebDriverWait y Expected Conditions: Utilizados para manejar esperas explícitas y mejorar la estabilidad de las pruebas.

* Google Chrome WebDriver: Se utiliza para ejecutar las pruebas en el navegador Chrome.
